# Audio Power Meter

Kuerzlich habe ich ein Monacor PM-100ST von 1975 repariert. 
Dazu habe ich den Schaltplan erstellt. 
 
Der Meter-Teil ist zweifach vorhanden (Links/Rechts). 
Der LED-Teil ist am linken Kanal angeschlossen. 
 
Die verwendeten Meter sind 100mV Zeigerinstrumente. 
Die Dioden sind aus Germanium. Da man diese heute kaum noch bekommt, 
kann man sie durch Schottky-Dioden ersetzen. 
 
Der Fehler in meinem Gerät war eine defekte D2. 
 
Die Diode in der LED-Schaltung kann auch eine Silizium sein, sie 
wird verwendet, dass die LED negative Spannung über ihr nicht sperren muss. 
Heutige LEDs dürften das aushalten, aber sicher ist sicher. 
 
D3 ist sehr wichtig zum Schutz der Zeigerinstrumete. Sie fängt ab 0,3V 
an zu leiten und schützt somit die Instrumente.
